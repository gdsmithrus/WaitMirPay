WaitServicePay

Приложение для смарт часов на базе wearos.

Предназначено, для предотвращения блокировки экрана, когда Вы поворачиваете руку. WaitServicePay позволяет упростить срабатывание датчика NFC для выполнения оплаты через приложение, которое требует включенный экран (например sberpay или mirpay).

Приложение проверено на часах от Galaxy Watch 5 pro и в эмуляторе.
